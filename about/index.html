About this site

This website is currently hosted on GitHub Pages. Previously, I ran themes made by other developers on static webpage generators , and what the user only need to do is to simply fill the theme with Markdown files. To develop my own theme, I taught myself a little HTML as the very first try. It turned out that CSS is really challenging for me, and I haven’t even touched JS yet…

As becoming more familiar with HTML, I started to write a skeleton for Hugo. Knowing nothing about GO language, I had to seek quite a bit of assistance from DeepSeek during my first front-end development. There remains an obvious distance between the skeleton and a real downloadable theme, which I hope I made in the future. I’ll call it SakuraChiyo when finished. By the way, I must say the official documents of Hugo are unfriendly organised.
